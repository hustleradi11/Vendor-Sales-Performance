Vendor Performance Analysis

Project Overview This project focuses on analyzing a vendor performance dataset to evaluate supplier efficiency and business performance. Using Python, the project applies data cleaning, exploratory data analysis (EDA), and visualization techniques to uncover meaningful insights and compare vendor metrics.

The analysis helps identify high-performing and low-performing vendors based on key business indicators. Objectives

Analyze vendor performance using structured business data
Identify trends and patterns across vendors
Compare performance metrics to distinguish top and low-performing vendors
Present insights through clear and effective visualizations
Technologies Used

Programming Language: Python
Libraries: Pandas, NumPy
Visualization Tools: Matplotlib, Seaborn
Environment: Jupyter Notebook
Key Tasks Performed

Cleaned and preprocessed the dataset to handle missing values and inconsistencies
Conducted Exploratory Data Analysis (EDA) to understand distributions and relationships
Analyzed vendor performance based on sales, pricing, and contribution metrics
Created visualizations to compare vendor trends and performance
Interpreted results to support data-driven decision making
Insights & Analysis

Identified vendors with strong and consistent performance
Analyzed variations in vendor pricing and contribution
Compared performance metrics to highlight areas of improvement
